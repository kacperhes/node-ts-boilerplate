## Project setup

1. Install dependencies

```shell
yarn
```
2. Run developement server

```shell
yarn dev
```
3. Build production code

```shell
yarn build
```

4. Run compiled code

```shell
yarn start
```

5. Run all unit tests

```shell
yarn test:unit
```
6. Lint code 

```shell
yarn lint
```

7. Format code

```shell
yarn format
```
